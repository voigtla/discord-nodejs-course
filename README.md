# Discord Bot制作講座：技術の裏側を理解する全10回

## この講座について

この講座は、**プログラミング未経験者**が「Discord Botがなぜ動くのか」を**理屈から理解する**ために設計されています。

単なるコピペではなく、**技術の裏側にある考え方**を学ぶことで、自分でコードを読み、改造し、問題を解決できる力を身につけます。

### 使用技術
- **Node.js**: JavaScriptをサーバー側で実行する環境
- **Discord.js**: Discord Botを作るためのライブラリ
- **SQLite**: データを記憶するための軽量データベース
- **Gemini API**: Google の生成AIを使うためのAPI

### この講座のゴール

全10回を終えたあなたは、以下ができるようになります：

- ✅ Discord botのコードを**怖がらずに読める**
- ✅ SQLiteを「なぜ使っているか」**論理的に説明できる**
- ✅ Gemini APIの構造を理解し、**自分の言葉で扱える**
- ✅ セキュリティ上の「地雷」を察知し、**自らブレーキをかけられる**

## 🚀 学習の進め方

### 1. 順番に読む
第1回から順に進めてください。後の回は、前の回の知識を前提としています。

### 2. コードは必ず手で書く
コピペだけでは理解できません。一文字ずつ打つことで、コードの「意味」が見えてきます。

### 3. 「なぜ?」を大切にする
この講座では、**「どう書くか」より「なぜそう書くか」**を重視しています。  
分からないことがあれば、立ち止まって考えてください。

### 4. 実際に動かす
各回の最後には、その時点での完成版コードがあります。必ず動かして、動作を確認してください。

- **APIキーやトークンは絶対に公開しない**（第9回で詳しく学びます）
